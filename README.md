Interactive Stock Ticker

Description:
An Interactive Stock Ticker System that simulates real-time stock prices, portfolio management, limit orders, and price alerts using Python.

Features:

Tracks top gainers and losers using heap data structures.

Allows users to place buy/sell orders and set price alerts.

Displays portfolio value, recent transactions, and market statistics.

Simulates market activity with AI-generated orders and real-time price updates.
